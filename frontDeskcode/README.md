# residential-property-management-system

#### 介绍
采用B/S结构的设计思想，利用Vue3框架构建前端界面，Express框架和Node.js运行环境搭建后端服务，选择MySQL作为数据库系统，同时以Visual Studio Code和微信开发者工具为主要开发工具，使用JavaScript、HTML、CSS编程语言，设计实现了一款适用于小区物业管理的系统。系统包括七大功能模块，具体为用户管理、公告管理、停车管理、报修管理、投诉管理、缴费管理以及访客管理。该系统支持Web端和小程序端两种访问方式，旨在覆盖小区物业管理员、员工及业主三类用户群体。

#### 安装教程
1.  npm install生成的node_modules文件夹
2.  npm i下载依赖
3.  npm run dev运行代码

如果下载了cnpm,可以使用下列代码，速度更快：
1.  cnpm insall生成的node_modules文件夹
2.  cnpm i下载依赖
3.  cnpm run dev运行代码

#### 使用说明

1.  使用后端给的定账号密码登录后台
2.  登录成功进入首页
